# Committees
Committees can be created, edited, and delete easily on the home page

## Creating Committees
To create a committee, first hit the "Create New Committee" button, and then enter a name for the committee. Hitting "Create" will then register a new committee with the server, and also update each client viewing the home page to show the newly created committee

## Editing Committees
Committees can be edited by pressing the orange gear icon seen next to its name on the home page. Doing so will allow the name to be edited, which will be synced across all clients

## Deleting Committees
A committee can be deleted by pressing the trash can icon next to its name on the home page, and then confirming the prompt asking if you wish to proceed
